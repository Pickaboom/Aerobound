AEROBOUND Website Starter

Files:
- index.html = main landing page
- support.html = support page
- privacy.html = starter privacy policy
- assets/style.css = website styling
- assets/script.js = FAQ click script

How to edit:
1. Replace # links with your real Meta Quest, Discord, TikTok, YouTube, and trailer links.
2. Replace support@aeroboundvr.com and privacy@aeroboundvr.com with emails you own.
3. Replace placeholder video boxes with YouTube iframe embeds.
4. Upload all files to GitHub Pages, Netlify, Vercel, or your web host.
5. Buy/connect a .com domain and point it to the host.
